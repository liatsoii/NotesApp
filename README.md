# Notes App - Flutter Web

Простое приложение для заметок.

## Функции
- Добавление заметок
- Редактирование заметок
- Удаление заметок
- Сохранение при обновлении приложения

## Структура проекта
lib/
- ├── main.dart          # Точка входа
- ├── models/            # Модели данных
- ├── screens/           # Экраны приложения
- ├── services/          # Сервисы
- └── widgets/           # Виджеты
